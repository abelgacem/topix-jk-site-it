# Definition
## Historically
- run a (guest) LINUX OS inside a (host) LINUX OS
- isolate processes that runs inside the guest os from the process that run in the host OS
- run a (guest aka WSL) LINUX OS inside a (host) WINDOWS OS

## Lexical field
- containerization
- isolation
