# Definition
- a **single** file@txt containing code@python
- named xxx.py
- code@python can be any numbers of
  - python:class
  - python:method
  - python:var
  - python:instruction


## example:
```
my_package/
├── __init__.py
├── module1.py
├── module2.py
└── subpackage/
    ├── __init__.py
    ├── module3.py
    └── module4.py
```
# Toknow
- The name of the file containing a module is the name of the module with the suffix .py appended
