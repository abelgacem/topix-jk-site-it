# The verb: *Is 1* and *Can be*
- denote 1 Type, 1 Category, 1 Classification
  - A is 1 B $\iff$ B can be 1 A

## Example
- 1 *elephant* is 1 *Animal*
- 1 *car* is 1 *Vehicle*
- 1 *truck* is 1 *Vehicle*
- 1 *Vehicle* can be 1 *car*
- 1 *Vehicle* can be 1 *truck*
