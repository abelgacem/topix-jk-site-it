# Whatis
- Display os:info
- set os:property 
# Get Info
```bash
# Get help
sudo systemsetup -help

# Get status ssh daemon
systemsetup -getremotelogin

```

# Set Property
```bash
# Allow ssh from remote
sudo systemsetup -setremotelogin on
```
